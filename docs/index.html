A simple yet powerful way to tag and search PDF files

What does it do?

#PDFSearch was originally developed to help researchers organise scientific literature. #PDFSearch searches the comments of your PDF files for hashtags. The hashtags can then be used to quickly filter and find the files which contain them.

All you have to do is add comments containing hashtags to your PDF files and #PDFSearch will do the rest.

Features

Simple

Just add Hashtags (e.g. #relevant2heartStudy) to your PDF files. #PDFSearch will find and index them.

Share

Because all the tags are part of the PDF file, when you share a file you also share the hashtags used by #PDFSearch.

Quick

Quickly filter a list of all your PDF files using the hashtags you have added to the PDF files comments.
